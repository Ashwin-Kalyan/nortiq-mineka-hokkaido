# Mineka Hokkaido Website

A modern, macOS-style website for Mineka Hokkaido private car hire and tour services.

## Features

- **macOS-style UI**: Rounded corners, frosted glass effects, and smooth animations
- **Red-White Gradient Accents**: Professional color scheme throughout
- **Frosted Glass Menubar**: Fixed top navigation with blur effects
- **Multi-language Support**: Japanese (default), English, and Simplified Chinese
- **No Buttons**: Clean interface using only menu navigation
- **Animated Footer**: Smooth entrance animation
- **Responsive Design**: Works on desktop and mobile devices

## Pages/Sections

1. **Home** - Hero section with welcome message
2. **Hire / Private Car Service** - Service details and features
3. **Tour Model Courses** - Four tour packages (Otaru/Sapporo, Furano/Biei, Lake Toya/Noboribetsu, Niseko)
4. **About Us** - Company introduction
5. **FAQ** - Frequently asked questions
6. **Service Areas** - Coverage areas in Hokkaido
7. **Contact** - Contact information and inquiry details

## Files

- `index.html` - Main HTML structure
- `styles.css` - All styling with macOS design system
- `script.js` - Language switching and animations

## Usage

Simply open `index.html` in a modern web browser. The website works entirely client-side with no build process required.

## Browser Support

Works best in modern browsers that support:
- CSS backdrop-filter
- CSS Grid
- ES6 JavaScript
